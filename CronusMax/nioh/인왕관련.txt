
자동 잔심 구현을 하면 RB를 더이상 누르지 않으므로

다음과 같이 설정한다.

RB + LB = 자동 잔심 온

RB + LT = 자동 잔심 오프

RB + LB + Y = 상단 시작 설정 -> 한번 시범 보임 (잔심 유전 2)
RB + LB + X = 중단 시작 설정 -> 한번 시범 보임 (잔심 유전 2)
RB + LB + A = 하단 시작 설정 -> 한번 시범 보임 (잔심 유전 2)
RB + LB + B로 변화 설정 : 도, 쌍도, 대태도, 창, 도끼, 사슬낫, 톤파 (각각에 대한 상, 중, 하단 스테미나 설정)

LB + Y, or X or A 시에 추적
